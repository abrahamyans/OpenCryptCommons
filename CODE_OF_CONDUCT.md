# Code of Conduct

## Our commitment

OpenCryptCommons aims to be a respectful, constructive, and inclusive open-source project.

Participants are expected to:

- communicate professionally and respectfully
- focus on technical substance and good-faith collaboration
- avoid harassment, personal attacks, or discriminatory language
- welcome constructive feedback and correction

## Scope

This Code of Conduct applies to repository discussions, issues, pull requests, and other project-related communications.

## Reporting

If you encounter behavior inconsistent with this Code of Conduct, please open a private communication channel with the project lead when available. Until then, use GitHub moderation and reporting tools where appropriate.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that violate this Code of Conduct.
